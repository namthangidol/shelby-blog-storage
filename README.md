# Shelby Blog Storage Demo

This project demonstrates how to use Shelby Protocol as decentralized storage for blog assets.

Example use case:
Nguyenamthang.com stores downloadable datasets, code examples, and tutorials on Shelby storage.

## Upload files
```bash
bash upload.sh
```

## Download files
```bash
bash download.sh
```

## Folder structure

blog-assets/
 - datasets
 - code
 - ebooks

## Shelby commands used
- shelby upload
- shelby download
- shelby info

This repository is a simple example of integrating Shelby storage into a content platform.
